# （three-way deadlock）

## ゲームの内容
（三竦み鬼ごっこ
RedチームはBlueチームに強く、触れると消滅させられる
BlueチームはYellowチームに強く、触れると消滅させられる
YellowチームはRedチームに強く、触れると消滅させられる
他のチームの敵がすべて消滅するか、制限時間まで逃げ切れば勝利
プレイヤー以外は自動で動き、自分が強いチームを追いかけ、自分が弱いチームから逃げる
各チーム5人、制限時間は40秒）
